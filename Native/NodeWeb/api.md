# 接口文档

| 接口                  | 类型   | 参数                        | 返回信息         |
| --------------------- | ----- |---------------------------- | --------------- |
| `register`: 注册      | post  | name(用户名)、password(密码) | status(状态)     |
| `login`: 登录         | post  | name(用户名)、password(密码) | status(状态)     |
